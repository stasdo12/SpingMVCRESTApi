Welcome to the API Page

This is an example of an API


/api/codes ::(GET)We can get all codes from binance
/api/codes/{title} ::(GET)We can get one code by title
/api/codes ::(PUT)We can add one code by title
/api/codes/{title} ::(DELETE)We can delete one code by title
/api/codes/{title} ::(PATCH)We can update one code by title
/api/codes/with/{title} ::(GET)We can get all codes from binance by Title
